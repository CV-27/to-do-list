# A simple task manager made in Vue3

## Dependencies:
- Pinia
- Pinia persist
- Supabase
- Bootstrap
- Vue-CLI

## Project premises
### Authentication
- Create an account
- Log in to the app
- Log out from the app

### App
- Add a new task
- Edit a task
- Mark a task as complete
- Mark a task as incomplete
- Delete a task

```
npm install
```

npm run serve
